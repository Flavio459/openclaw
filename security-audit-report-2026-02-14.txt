Warnings: 1

1. gateway.trusted_proxies_missing - Reverse proxy headers are not trusted
   Descrição: O gateway.bind está configurado como loopback e gateway.trustedProxies está vazio. Se você expor a Control UI através de um reverse proxy, configure proxies confiáveis para evitar spoofing de checks de cliente local.
   Impacto: Possível bypass de verificações de segurança se a UI for exposta via proxy sem configuração adequada.
   Correção: Defina gateway.trustedProxies com os IPs do seu proxy ou mantenha a Control UI apenas local.
   Comando sugerido: Editar configuração do OpenClaw (openclaw.json) e adicionar "gateway": { "trustedProxies": ["IP_DO_PROXY"] }.
